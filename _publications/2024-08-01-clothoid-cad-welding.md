This work, originating from my Master's thesis, presents a robust motion planning workflow for robotic welding based on the Learning by Demonstration paradigm. A human operator teaches the robot the desired welding trajectory using a hand-tool pointer tracked by an optical system, which is then compensated against the workpiece CAD model using a clothoid-based approach for improved precision. The experimental collaborative welding cell integrates a Franka Emika Panda robotic manipulator with an OptiTrack motion capture system, with the full software stack developed in C++ and ROS.
