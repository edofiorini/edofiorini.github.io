Development of an autonomous rover for precision agriculture, specifically designed for a range of field tasks including navigating fields, weed removal, picking fruits, and visual inspection.

* Developed navigation and control systems using ROS/ROS2, implementing SLAM for autonomous field mapping and localization, alongside the ROS2 Navigation stack for path planning and obstacle avoidance
* Integrated GPS-based positioning to enable outdoor navigation and control across large agricultural fields
* Worked with a UR5 robotic manipulator for task execution such as weed removal and fruit picking
* Implemented perception pipelines combining Intel RealSense and ZED stereo cameras with YOLOv6 for real-time object detection, complemented by a custom-built, in-house multispectral camera for enhanced environmental perception
* Developed front-end visualization tools using JavaScript and Vue.js
* Validated the platform's performance in simulation environments (CoppeliaSim and Gazebo) and in real-world field navigation tests

Carried out as part of my role as Robotics Researcher at the ALTAIR Robotics Lab, University of Verona.
